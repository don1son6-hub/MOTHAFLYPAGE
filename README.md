# MOTHa Fly Managing & Holding Company – Landing Page

Simple, straight-to-the-point landing page built to:

- Show partners web development and structuring capabilities.
- Present MOTHa Fly’s mission and core services.
- Capture free consultation requests and send them via email.

## How it works

1. Open `index.html` in a browser or host via GitHub Pages.
2. Update `script.js`:
   - Replace `YOUR_EMAIL_HERE` with the email where you want notifications.
3. When a visitor submits the consultation form:
   - Their default mail app opens.
   - An email is pre-filled with their details.
   - They send it, and you receive the request.

## Files

- `index.html` – main page
- `styles.css` – styling
- `script.js` – form logic and email notification
- `README.md` – project overview
